# Tunnel Next

## 概述

Tunnel Next 是一个基于节点的图像处理应用程序。它提供了一个节点图编辑器以及可由用户自行编写的Python节点脚本，允许高度复杂的、非破坏性的、可定制的图像处理流程。在未来，本应用程序会支持色彩管理和频域图像处理等功能，进一步扩展其能力。

## 如何运行

1.  确保您已安装所有必需的库（见“系统要求”）。
2.  下载或克隆此仓库。
3.  在终端中，导航到包含 `Tunnel Next.py` 文件的目录。
4.  运行以下命令启动应用程序：

    ```shell
    python "Tunnel Next.py"
    ```

## 文件结构 (主要)

*   `Tunnel Next.py`: 应用程序的主要 Python 脚本。
*   `resources/`: 存放主题文件 (`.TNXtheme`) 和其他资源的目录。
*   `scripts/`: 存放用户自定义处理脚本的目录。
